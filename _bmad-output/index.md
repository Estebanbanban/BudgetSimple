## Project Overview

Budgetsimple is a local-first budget & investing tracker that runs entirely in the browser. It supports CSV import (transactions + income), categorization rules, budgets, envelope-style savings goals, dashboards/charts, and exports — with all data stored locally (IndexedDB + localStorage).

## Quick Reference

- **Tech Stack:** Static HTML/CSS + vanilla JS (no build step)
- **Entry Point:** `index.html` → `app.js`
- **Storage:**
  - IndexedDB database `budgetsimple` (schema v2): `transactions`, `income`, `envelopes`, `envelopeContribs`, `meta`
  - localStorage config key: `budgetsimple:v1` (categories, rules, budgets, settings)
  - Fallback localStorage store prefix: `budgetsimple:fallback:*` (if IndexedDB unavailable)

## Getting Started

### Run Locally

```bash
python3 -m http.server 8080
```

Open `http://localhost:8080` and use **Import** with `samples/transactions.csv` (and optionally `samples/income.csv`).
